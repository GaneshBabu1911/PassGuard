How K-Anonymity Works

1
Your browser SHA-1 hashes the password locally
2
Only the first 5 characters of the hash are sent to api.pwnedpasswords.com
3
The API returns all hashes starting with those 5 chars (~400 results)
4
We check locally if the remaining 35 chars appear in the list
